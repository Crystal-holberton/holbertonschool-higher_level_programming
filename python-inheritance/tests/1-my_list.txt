Use of MyList
=============

''print_sorted()'' returns the list sorted

positive integers
>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> isinstance(my_list, list)
True
>>> my_list.print_sorted()
[]

Check that appending works:
>>> my_list.append(1)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(4)
>>> my_list.print_sorted()
[1, 2, 3, 4]

Check unsorted appending:
>>> my_list = MyList()
>>> my_list.append(5)
>>> my_list.append(3)
>>> my_list.append(4)
>>> my_list.append(1)
>>> my_list.append(2)
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

Negative numbers:
>>> my_list.append(-5)
>>> my_list.print_sorted()
[-5, 1, 2, 3, 4, 5]

Check empty list:
>>> my_list = MyList()
>>> my_list.print_sorted()
[]

Check original list is not modified:
>>> my_list = MyList()
>>> my_list.append(3)
>>> my_list.append(1)
>>> my_list.append(2)
>>> sorted_list = my_list.print_sorted()
print(my_list)
[1, 2, 3]
