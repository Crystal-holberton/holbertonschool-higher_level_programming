Use of MyList
=============

''print_sorted()'' returns the list sorted
positive integers
>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> isinstance(my_list, list)
True
>>> my_list.print_sorted()
[]

Check that appending works:
>>> my_list.append(1)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(4)
>>> my_list.print_sorted()
[1, 2, 3, 4]


Negative numbers:
>>> my_list.append(-5)
>>> my_list.print_sorted()
[-5, 1, 2, 3, 4]

None value:
>>> my_list.append(None)
>>> my_list.print_sorted()
Traceback (most recent call last):
TypeError: unorderable types: NoneType() < int()

NULL arguments:
>>> my_list.append(NULL)
Traceback (most recent call last):
NameError: name 'NULL' is not defined
